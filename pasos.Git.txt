git init //inicia un retpositorio
la U significa untracked o sea sin seguimineto
git add . //le da seguimiento a todos los archivos de mi retpositorio, los prepara para ser guardados
git add + archivo especifico // le da seguimiento al archivo especificado
creando una carpeta .gitignore dentro de ella espesificamos los nombres de los archivos que queremos ignorar parea que no se suban a mi repo
git commit _m "una comentario tipo version" // esto sube y crea una version de mi proyecto
